# London Black Lives Matter Community

### Central community base for news and updates

### To run the project

- \$ git clone https://github.com/habin-isa/ldn-blm-community.git
- \$ cd ldn-blm-community
- \$ yarn
- \$ yarn start

- \$ yarn test
